# TidyTuesday / PydyTuesday 

Weekly data wrangling and visualization exercises in Python, working through
the [TidyTuesday](https://github.com/rfordatascience/tidytuesday) dataset 
series. Started with week 02 in January 2026 and working to catch up to weekly releases. 

## Background

I'm an epidemiologist that uses R by day but wanting to branch out into Python. This repo documents a learning project through TidyTuesday (PydyTuesday) dataset exploration. The goal of this project is to learn python skills related to data wrangling and visualization. 

## Structure

Projects are organized by year. Each folder contains a code (py scripts or Jupyter notebooks) with code and outputs. 

```
2026/
  week_01_dataset_name/
  week_02_dataset_name/
  ...
```

## Tools

Python · pandas · matplotlib · seaborn · plotly · Jupyter
